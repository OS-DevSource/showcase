# Polish checklist

Pass 1 - Typography and spacing

- [x] Headings use the display font consistently
- [x] Body copy uses a single scale and balanced line lengths
- [x] Section spacing is consistent

Pass 2 - Mobile sweep

- [x] Hero spacing and CTA stack tuned for small screens
- [x] Gallery tiles and cards stack cleanly
- [x] Proof slider layout remains stable

Pass 3 - Accessibility and performance

- [x] Focus states are visible on all controls
- [x] Reduced motion disables key effects
- [x] No heavy media or layout shift risks
